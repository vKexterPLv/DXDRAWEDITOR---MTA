# KEYBINDINGS

| KEY  | WHAT ITS DOING |
| ------------- | ------------- |
| K  | RESIZE |
| ARROWS+SHIFT  | MOVING ELEMENT FASTER  |
| ARROWS | MOVING ELEMENT |
| S | RESIZE |
| R | ROTATION |


Keybind `K` is for resizing using red points and `S` for resizing width and height at the same time while our element is selected
